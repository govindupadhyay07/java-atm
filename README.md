# Java ATM Machine

## Description
A simple console-based ATM simulation developed using Java.

## Features
- Check Balance
- Deposit Money
- Withdraw Money
- Exit
- Random initial account balance

## Concepts Used
- Java
- Scanner Class
- Random Class
- switch statement
- do-while loop

## How to Run

Compile:
```bash
javac Atm.java
```

Run:
```bash
java Atm
```

## Future Improvements
- PIN authentication
- Insufficient balance check
- Transaction history
- Multiple user accounts
